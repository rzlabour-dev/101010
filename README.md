# PDF to Image Converter

A client-side PDF to image converter that works entirely in your browser. No file uploads to external servers required.

## Features

- **Privacy Focused**: All processing happens locally in your browser
- **Smart Export**: Single page PDFs download as JPG, multi-page as ZIP
- **Multiple Formats**: Export as JPG, PNG, or WebP
- **Quality Control**: Adjust image quality and resolution
- **Responsive Design**: Works on desktop and mobile devices

## How to Use

1. Open `index.html` in your browser
2. Drag & drop a PDF file or click to browse
3. Select your preferred image format, quality, and resolution
4. Click "Convert PDF to Images"
5. Download individual images or all images as a ZIP
